### Tabla: Cliente

| Parámetro                         | Valor                                                        |
| --------------------------------- | ------------------------------------------------------------ |
| Tamaño de bloque                  | 8 KB (8,192 bytes)                                           |
| Tamaño de extent                  | ---                                                          |
| Estructura del registro lógico    | 461 bytes                                                    |
| Overhead de fila                  | 23 bytes                                                     |
| Filas por bloque (estimado)       | 13                                                           |
| Espacio libre reservado           | `ALTER TABLE MODULO_RESERVAS.Cliente SET (FILLFACTOR = 80);` |
| **Tamaño total estimado (año 3)** | **≈ 729,088 bytes (≈ 712 KB)**                               |

---

**Cálculos**

- **Espacio libre por bloque:**  
(8,192 − 24) × 0.8 = **6,534.4 bytes por bloque**

- **Tamaño por fila con overhead:**  
461 + 23 = **484 bytes**

- **Filas por bloque:**  
6,534.4 ÷ 484 ≈ 13.51 ≈ **13 filas por bloque**

- **Cantidad de filas (año 3):**  
**1,156 filas**

**Cantidad de bloques necesarios:**  
1,156 ÷ 13 ≈ 88.92 ≈ **89 bloques**

**Tamaño total estimado:**  
89 × 8,192 = **729,088 bytes ≈ 712 KB**

## Módulo 5 - Módulo de Monitoreo de Entrega:

### Tabla: Notificacion

| **Parámetro** | **Valor** |
| -- | -- |
| Tamaño de bloque | 8 KB (8,192 bytes) |
| Tamaño de extent | 1 MB (128 bloques) |
| Tamaño registro lógico | 69 bytes |
| Tamaño registro físico | 92 bytes |
| FILLFACTOR | 90% |
| Espacio libre reservado | 10% |
| Registros por bloque | 71 |
| Proyección 2028 | 77.2 GB |

---

```sql
Espacio Libre por Bloque = (Tamaño de Bloque - Sobrecarga de Bloque) × Factor
Espacio Libre por Bloque = (8,192 - 24) × 0.90
Espacio Libre por Bloque = 7,351.2 × 0.90
Espacio Libre por Bloque = 6,616.08 bytes

Cantidad de Filas por Bloque = Espacio Libre por Bloque / (Tamaño de Fila + Sobrecarga de Fila)
Cantidad de Filas por Bloque = 6,616.08 / (69 + 23)
Cantidad de Filas por Bloque = 6,616.08 / 92
Cantidad de Filas por Bloque = 71.91 ≈ 71 filas

Cantidad de Bloques = Cantidad Total de Filas / Cantidad de Filas por Bloque

Año 2025:
Cantidad de Bloques = 105,120,000 / 71
Cantidad de Bloques = 1,480,563 bloques
Tamaño Total = Cantidad de Bloques × Tamaño de Bloque
Tamaño Total = 1,480,563 × 8,192 bytes
Tamaño Total = 12,128,772,096 bytes
Tamaño Total = 12,128,772,096 / (1024³) GB
Tamaño Total = 11.297 GB ≈ 11.3 GB
```

---
