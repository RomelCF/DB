# 10.3.5. Módulo 5 / Integrante 5

## Módulo de Monitoreo de Entrega

### Creación de secuencias

1. **Secuencia 1:** seq_operacion_codigo
    - **Propósito:** Generar códigos únicos para operaciones
    - **Formato:** OP-YYYY-NNNNNN

    **Código de creación de secuencia**

```sql
-- Crear la secuencia
CREATE SEQUENCE monitoreo.seq_operacion_codigo
    START WITH 1
    INCREMENT BY 1
    MINVALUE 1
    MAXVALUE 999999
    CYCLE
    CACHE 20;

-- Establecer el valor actual basado en operaciones existentes
DO $$
DECLARE
    v_max_numero INTEGER;
BEGIN
    -- Extraer el número más alto de los códigos existentes
    SELECT COALESCE(
        MAX(
            CAST(
                SUBSTRING(codigo FROM '[0-9]+$') AS INTEGER
            )
        ), 0
    ) INTO v_max_numero
    FROM monitoreo.Operacion 
    WHERE codigo ~ '^OP-[0-9]{4}-[0-9]{6}$';
    
    -- Establecer la secuencia al siguiente valor
    PERFORM setval('monitoreo.seq_operacion_codigo', v_max_numero + 1);
    
    RAISE NOTICE '  ✓ Secuencia inicializada en: %', v_max_numero + 1;
END $$;
```

2. **Secuencia 2:** seq_entrega_codigo
    - **Propósito:** Generar códigos únicos para entregas
    - **Formato:** ENT-YYYY-NNNNNN

    **Código de creación de secuencia**

```sql
-- Crear la secuencia
CREATE SEQUENCE monitoreo.seq_entrega_codigo
    START WITH 1
    INCREMENT BY 1
    MINVALUE 1
    MAXVALUE 999999
    CYCLE
    CACHE 10;

-- Establecer el valor actual basado en entregas existentes

DECLARE

```

### Funciones de ayuda

```sql
-- Función para generar código de operación
CREATE OR REPLACE FUNCTION monitoreo.generar_codigo_operacion()
RETURNS VARCHAR(20)
LANGUAGE plpgsql
AS $$
DECLARE
    v_codigo VARCHAR(20);
    v_numero INTEGER;
    v_anio VARCHAR(4);
BEGIN
    -- Obtener el año actual
    v_anio := EXTRACT(YEAR FROM CURRENT_DATE)::VARCHAR;
    
    -- Obtener el siguiente número de la secuencia
    v_numero := nextval('monitoreo.seq_operacion_codigo');
    
    -- Generar el código con formato OP-YYYY-NNNNNN
    v_codigo := 'OP-' || v_anio || '-' || LPAD(v_numero::TEXT, 6, '0');
    
    RETURN v_codigo;
END;
$$;

-- Función para generar código de entrega
CREATE OR REPLACE FUNCTION monitoreo.generar_codigo_entrega()
RETURNS VARCHAR(20)
LANGUAGE plpgsql
AS $$
DECLARE
    v_codigo VARCHAR(20);
    v_numero INTEGER;
    v_anio VARCHAR(4);
BEGIN
    -- Obtener el año actual
    v_anio := EXTRACT(YEAR FROM CURRENT_DATE)::VARCHAR;
    
    -- Obtener el siguiente número de la secuencia
    v_numero := nextval('monitoreo.seq_entrega_codigo');
    
    -- Generar el código con formato ENT-YYYY-NNNNNN
    v_codigo := 'ENT-' || v_anio || '-' || LPAD(v_numero::TEXT, 6, '0');
    
    RETURN v_codigo;
END;
$$;
```

### Scripts de ejemplo

1. Insertar nueva operación con código automático:

```sql
INSERT INTO monitoreo.Operacion (id_operacion, codigo, fecha_inicio, id_estado_operacion)
    VALUES (
        gen_random_uuid(),
        monitoreo.generar_codigo_operacion(),  -- Genera: OP-2025-000001
        CURRENT_TIMESTAMP,
        (SELECT id_estado_operacion FROM monitoreo.EstadoOperacion WHERE nombre = 'En curso')
    );
```

2. Insertar nueva entrega con código automático:

```sql
INSERT INTO monitoreo.Entrega (id_entrega, codigo, fecha_entrega, lugar_entrega, id_estado_entrega, id_contenedor, id_importador)
    VALUES (
        gen_random_uuid(),
        monitoreo.generar_codigo_entrega(),  -- Genera: ENT-2025-000001'
        CURRENT_TIMESTAMP,
        'Av. Industrial 123, Lima',
        (SELECT id_estado_entrega FROM monitoreo.EstadoEntrega WHERE nombre = 'Completada'),
        (SELECT id_contenedor FROM monitoreo.Contenedor WHERE codigo = 'CONT-001'),
        (SELECT id_importador FROM monitoreo.Importador WHERE codigo = 'IMP-001')
    );
```
