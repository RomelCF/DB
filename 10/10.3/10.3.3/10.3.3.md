# 10.3.3. Módulo 3 / Integrante 3

## Módulo de Gestión de Operaciones Marítimas 

### Creación de secuencias

### 1. **Secuencia para Operacion (OP-YYYY-NNN)**
```sql
-- Secuencia para códigos de Operacion
CREATE SEQUENCE gestion_maritima.seq_operacion_codigo
    START WITH 1
    INCREMENT BY 1
    MINVALUE 1
    MAXVALUE 999
    CYCLE
    CACHE 10;

-- Inicializar secuencia basada en datos existentes
DO $$
DECLARE
    v_max_numero INTEGER;
BEGIN
    SELECT COALESCE(
        MAX(CAST(SUBSTRING(codigo FROM '[0-9]+$') AS INTEGER)), 0
    ) INTO v_max_numero
    FROM gestion_maritima.Operacion 
    WHERE codigo ~ '^OP-[0-9]{4}-[0-9]{3}$';
    
    PERFORM setval('gestion_maritima.seq_operacion_codigo', v_max_numero + 1);
    RAISE NOTICE 'Secuencia operacion inicializada en: %', v_max_numero + 1;
END $$;
```

## **Funciones de Generación de Códigos**

### 1. **Función para código de Operacion**
```sql
CREATE OR REPLACE FUNCTION gestion_maritima.generar_codigo_operacion()
RETURNS VARCHAR(20)
LANGUAGE plpgsql
AS $$
DECLARE
    v_codigo VARCHAR(20);
    v_numero INTEGER;
    v_anio VARCHAR(4);
BEGIN
    v_anio := EXTRACT(YEAR FROM CURRENT_DATE)::VARCHAR;
    v_numero := nextval('gestion_maritima.seq_operacion_codigo');
    v_codigo := 'OP-' || v_anio || '-' || LPAD(v_numero::TEXT, 3, '0');
    RETURN v_codigo;
END;
$$;
```

## **Ejemplos de Uso**

### 1. **Insertar nueva Operacion**
```sql
INSERT INTO gestion_maritima.Operacion (id_operacion, codigo, fecha_inicio, id_estado_operacion)
VALUES (
    gen_random_uuid(),
    gestion_maritima.generar_codigo_operacion(),  -- Ahora genera: OP-2025-001
    CURRENT_TIMESTAMP,
    (SELECT id_estado_operacion FROM gestion_maritima.EstadoOperacion WHERE nombre = 'En Planificación')
);
```

### 2. **Insertar nueva OperacionMaritima**
```sql
INSERT INTO gestion_maritima.OperacionMaritima (id_operacion_maritima, id_operacion, codigo, cantidad_contenedores, id_estatus_navegacion, porcentaje_trayecto, id_buque)
VALUES (
    gen_random_uuid(),
    'id_operacion_existente',
    gestion_maritima.generar_codigo_operacion_maritima(),  -- Genera: OPM-2025-001
    150,
    (SELECT id_estatus_navegacion FROM gestion_maritima.EstatusNavegacion WHERE nombre = 'En Puerto'),
    0.00,
    'id_buque_existente'
);
```

### 3. **Insertar nueva OperacionPortuaria**
```sql
INSERT INTO gestion_maritima.OperacionPortuaria (id_operacion_portuaria, id_operacion, codigo, id_puerto, id_muelle, id_tipo_operacion_portuaria, id_buque)
VALUES (
    gen_random_uuid(),
    'id_operacion_existente',
    gestion_maritima.generar_codigo_operacion_portuaria(),  -- Genera: OPP-2025-001
    'id_puerto_existente',
    'id_muelle_existente',
    (SELECT id_tipo_operacion_portuaria FROM gestion_maritima.TipoOperacionPortuaria WHERE nombre = 'Carga'),
    'id_buque_existente'
);
```

### 4. **Insertar nueva Ruta**
```sql
INSERT INTO gestion_maritima.Ruta (id_ruta, codigo, origen, destino, duracion, tarifa)
VALUES (
    gen_random_uuid(),
    gestion_maritima.generar_codigo_ruta(),  -- Genera: RUTA-001
    'Callao, Perú',
    'Shanghai, China',
    25,
    15000.00

```
