# 10.4.3. Módulo 3 / Integrante 3

## Módulo de Gestión de Operaciones Marítimas

- **Nota:** Utilizar los [scripts de creación de tablas para este módulo](/scripts/ddl/tablas/integrante3.sql) y el [poblamiento inicial de datos](/scripts/dml/datos-iniciales/integrante3.sql) o en su defecto el poblamiento masivo de datos para este módulo.


### Creación de vistas materializadas

### P05 — Operaciones marítimas (Dashboard)

#### Vista: `vm_operaciones_en_curso`

**Descripción:** Estado actual de todas las operaciones activas (en progreso, planificación, etc.)

```sql
CREATE MATERIALIZED VIEW gestion_maritima.vm_operaciones_en_curso AS
SELECT 
    o.id_operacion,
    o.codigo AS codigo_operacion,
    COALESCE(om.cantidad_contenedores, 0) AS cantidad_contenedores,
    en.nombre AS estatus_navegacion,
    b.matricula AS matricula_buque,
    b.nombre AS nombre_buque,
    eo.nombre AS estado_operacion,
    o.fecha_inicio,
    o.fecha_fin
FROM 
    gestion_maritima.Operacion o
    INNER JOIN gestion_maritima.EstadoOperacion eo 
        ON o.id_estado_operacion = eo.id_estado_operacion
    LEFT JOIN gestion_maritima.OperacionMaritima om 
        ON o.id_operacion = om.id_operacion
    LEFT JOIN gestion_maritima.Buque b 
        ON om.id_buque = b.id_buque
    LEFT JOIN gestion_maritima.EstatusNavegacion en 
        ON om.id_estatus_navegacion = en.id_estatus_navegacion
WHERE 
    eo.nombre IN ('En Planificación', 'En Progreso', 'En Espera');
```



- **Comprobamos la vista materializada:**
```sql
select * from gestion_maritima.vm_operaciones_en_curso;
```

### P06 — Registro de Operación Marítima (Detalle)

#### Vista: `vm_operacion_detalle_completo`

**Descripción:** Detalle completo de una operación con buque, ruta, tripulación, muelle y contenedores.

```sql
CREATE MATERIALIZED VIEW gestion_maritima.vm_operacion_detalle_completo AS
SELECT 
    o.id_operacion,
    o.codigo AS codigo_operacion,
    b.nombre AS nombre_buque,
    b.matricula AS matricula_buque,
    rm.id_puerto_origen,
    rm.id_puerto_destino,
    rm.distancia,
    op.id_muelle,
    m.codigo AS codigo_muelle,
    COUNT(DISTINCT oc.id_contenedor) AS total_contenedores,
    COUNT(DISTINCT oe.id_empleado) AS total_tripulacion
FROM 
    gestion_maritima.Operacion o
    LEFT JOIN gestion_maritima.OperacionMaritima om 
        ON o.id_operacion = om.id_operacion
    LEFT JOIN gestion_maritima.Buque b 
        ON om.id_buque = b.id_buque
    LEFT JOIN gestion_maritima.OperacionRutaMaritima orm 
        ON om.id_operacion_maritima = orm.id_operacion_maritima
    LEFT JOIN gestion_maritima.RutaMaritima rm 
        ON orm.id_ruta_maritima = rm.id_ruta_maritima
    LEFT JOIN gestion_maritima.OperacionPortuaria op 
        ON o.id_operacion = op.id_operacion
    LEFT JOIN gestion_maritima.Muelle m 
        ON op.id_muelle = m.id_muelle
    LEFT JOIN gestion_maritima.OperacionContenedor oc 
        ON o.id_operacion = oc.id_operacion
    LEFT JOIN gestion_maritima.OperacionEmpleado oe 
        ON o.id_operacion = oe.id_operacion
GROUP BY 
    o.id_operacion, o.codigo, b.nombre, b.matricula, 
    rm.id_puerto_origen, rm.id_puerto_destino, rm.distancia,
    op.id_muelle, m.codigo;
```

- **Comprobamos la vista materializada:**
```sql
select * from gestion_maritima.vm_operacion_detalle_completo;
```

### P12 — Gestión de Certificaciones Aduaneras

#### Vista: `vm_certificaciones_asignadas`

**Descripción:** Certificaciones asignadas a operaciones con su estado y país.

```sql
CREATE MATERIALIZED VIEW gestion_maritima.vm_certificaciones_asignadas AS
SELECT 
    om.id_operacion_maritima,
    om.codigo AS codigo_operacion_maritima,
    ca.id_certificacion_aduanera,
    ca.nombre AS certificacion,
    ca.pais_aplicacion AS pais_certificacion,
    oca.fecha_aprobacion,
    oca.estado AS estado_aprobacion
FROM 
    gestion_maritima.OperacionMaritima om
    INNER JOIN gestion_maritima.OperacionCertificacionAduanera oca 
        ON om.id_operacion_maritima = oca.id_operacion_maritima
    INNER JOIN gestion_maritima.CertificacionAduanera ca 
        ON oca.id_certificacion_aduanera = ca.id_certificacion_aduanera;
```

- **Comprobamos la vista materializada:**
```sql
select * from gestion_maritima.vm_certificaciones_asignadas;
```

### P15 — Operaciones con Incidencias

#### Vista: `vm_incidencias_activas`

**Descripción:** Incidencias abiertas con tipo, gravedad y si tienen investigación.

```sql
CREATE MATERIALIZED VIEW gestion_maritima.vm_incidencias_activas AS
SELECT 
    i.id_incidencia,
    i.codigo AS codigo_incidencia,
    o.codigo AS codigo_operacion,
    b.nombre AS nombre_buque,
    ti.nombre AS tipo_incidencia,
    i.grado_severidad AS gravedad,
    i.fecha_hora AS fecha_reporte,
    i.descripcion
FROM 
    gestion_maritima.Incidencia i
    INNER JOIN gestion_maritima.Operacion o 
        ON i.id_operacion = o.id_operacion
    INNER JOIN gestion_maritima.TipoIncidencia ti 
        ON i.id_tipo_incidencia = ti.id_tipo_incidencia
    LEFT JOIN gestion_maritima.OperacionMaritima om 
        ON o.id_operacion = om.id_operacion
    LEFT JOIN gestion_maritima.Buque b 
        ON om.id_buque = b.id_buque
WHERE 
    i.descripcion NOT ILIKE '%resuelta%' 
    AND i.descripcion NOT ILIKE '%cerrada%';
```

### P16 — Inspecciones Realizadas

#### Vista: `vm_inspecciones_con_hallazgos`

**Descripción:** Inspecciones realizadas con prioridad y hallazgos asociados.

```sql
CREATE MATERIALIZED VIEW gestion_maritima.vm_inspecciones_con_hallazgos AS
SELECT 
    ins.id_inspeccion,
    ins.codigo AS codigo_inspeccion,
    o.codigo AS codigo_operacion,
    ti.nombre AS tipo_inspeccion,
    ei.nombre AS estado_inspeccion,
    p.nombre AS prioridad,
    COUNT(h.id_hallazgo) AS total_hallazgos,
    STRING_AGG(th.nombre || ': ' || h.descripcion, '; ') AS resumen_hallazgos
FROM 
    gestion_maritima.Inspeccion ins
    INNER JOIN gestion_maritima.Operacion o 
        ON ins.id_operacion = o.id_operacion
    INNER JOIN gestion_maritima.TipoInspeccion ti 
        ON ins.id_tipo_inspeccion = ti.id_tipo_inspeccion
    INNER JOIN gestion_maritima.EstadoInspeccion ei 
        ON ins.id_estado_inspeccion = ei.id_estado_inspeccion
    LEFT JOIN gestion_maritima.Prioridad p 
        ON ins.id_prioridad = p.id_prioridad
    LEFT JOIN gestion_maritima.Hallazgo h 
        ON ins.id_inspeccion = h.id_inspeccion
    LEFT JOIN gestion_maritima.TipoHallazgo th 
        ON h.id_tipo_hallazgo = th.id_tipo_hallazgo
GROUP BY 
    ins.id_inspeccion, ins.codigo, o.codigo, 
    ti.nombre, ei.nombre, p.nombre;
```
