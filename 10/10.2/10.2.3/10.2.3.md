# 10.2.3. Módulo 3 / Integrante 3

## Módulo de Gestión de Operaciones Marítimas

- **Nota:** Utilizar los [scripts de creación de tablas para este módulo](/scripts/ddl/tablas/integrante3.sql) y el [poblamiento inicial de datos](/scripts/dml/datos-iniciales/integrante3.sql) o en su defecto el poblamiento masivo de datos para este módulo.


### Creación de vistas

## 1. Vista: v_dashboard_operaciones

**Pantalla:** P1 - Dashboard de Operaciones Portuarias y P5 - Operaciones Marítimas

**Descripción:** Proporciona un resumen ejecutivo de las operaciones portuarias y marítimas con contadores agrupados por estado, métricas de embarcaciones y listado principal de operaciones.

**Tablas incluidas:** Operacion, EstadoOperacion, OperacionPortuaria, OperacionMaritima, Buque, Puerto, Muelle, TipoOperacionPortuaria, EstatusNavegacion

### Script de creación de vista

```sql
CREATE OR REPLACE VIEW gestion_maritima.v_dashboard_operaciones AS
SELECT 
    -- Métricas generales de operaciones
    COUNT(DISTINCT o.id_operacion) AS total_operaciones,
    COUNT(DISTINCT CASE WHEN eo.nombre = 'En Progreso' THEN o.id_operacion END) AS operaciones_en_curso,
    COUNT(DISTINCT CASE WHEN eo.nombre = 'Completada' THEN o.id_operacion END) AS operaciones_completadas,
    COUNT(DISTINCT CASE WHEN eo.nombre = 'En Planificación' THEN o.id_operacion END) AS operaciones_pendientes,
    COUNT(DISTINCT CASE WHEN eo.nombre = 'Cancelada' THEN o.id_operacion END) AS operaciones_canceladas,
    
    -- Métricas de operaciones portuarias
    (SELECT COUNT(*) FROM gestion_maritima.OperacionPortuaria) AS total_operaciones_portuarias,
    (SELECT COUNT(*) FROM gestion_maritima.OperacionPortuaria op
     JOIN gestion_maritima.Operacion o2 ON op.id_operacion = o2.id_operacion
     JOIN gestion_maritima.EstadoOperacion eo2 ON o2.id_estado_operacion = eo2.id_estado_operacion
     WHERE eo2.nombre = 'En Progreso') AS operaciones_portuarias_en_curso,
    
    -- Métricas de operaciones marítimas
    (SELECT COUNT(*) FROM gestion_maritima.OperacionMaritima) AS total_operaciones_maritimas,
    (SELECT COUNT(*) FROM gestion_maritima.OperacionMaritima om
     JOIN gestion_maritima.Operacion o2 ON om.id_operacion = o2.id_operacion
     JOIN gestion_maritima.EstadoOperacion eo2 ON o2.id_estado_operacion = eo2.id_estado_operacion
     WHERE eo2.nombre = 'En Progreso') AS operaciones_maritimas_en_curso,
    
    -- Métricas de embarcaciones
    (SELECT COUNT(*) FROM gestion_maritima.Buque) AS total_embarcaciones,
    (SELECT COUNT(*) FROM gestion_maritima.Buque b
     JOIN gestion_maritima.EstadoEmbarcacion ee ON b.id_estado_embarcacion = ee.id_estado_embarcacion
     WHERE ee.nombre = 'En operación') AS embarcaciones_operativas,
    (SELECT COUNT(*) FROM gestion_maritima.Buque b
     JOIN gestion_maritima.EstadoEmbarcacion ee ON b.id_estado_embarcacion = ee.id_estado_embarcacion
     WHERE ee.nombre = 'En mantenimiento') AS embarcaciones_en_mantenimiento,
    
    -- Métricas de contenedores
    (SELECT COUNT(*) FROM gestion_maritima.Contenedor) AS total_contenedores,
    (SELECT SUM(om.cantidad_contenedores) 
     FROM gestion_maritima.OperacionMaritima om
     JOIN gestion_maritima.Operacion o2 ON om.id_operacion = o2.id_operacion
     JOIN gestion_maritima.EstadoOperacion eo2 ON o2.id_estado_operacion = eo2.id_estado_operacion
     WHERE eo2.nombre = 'En Progreso') AS contenedores_en_transito,
    
    -- Duración promedio de operaciones completadas
    (SELECT AVG(EXTRACT(EPOCH FROM (o2.fecha_fin - o2.fecha_inicio)) / 3600)::DECIMAL(10,2)
     FROM gestion_maritima.Operacion o2
     JOIN gestion_maritima.EstadoOperacion eo2 ON o2.id_estado_operacion = eo2.id_estado_operacion
     WHERE eo2.nombre = 'Completada' AND o2.fecha_fin IS NOT NULL) AS duracion_promedio_horas,
    
    -- Timestamp de consulta
    CURRENT_TIMESTAMP AS fecha_consulta
    
FROM gestion_maritima.Operacion o
JOIN gestion_maritima.EstadoOperacion eo ON o.id_estado_operacion = eo.id_estado_operacion;
```

## 2. Vista: v_listado_operaciones

**Pantalla:** P1 - Dashboard de Operaciones Portuarias (tabla de listado) y P5 - Operaciones Marítimas

**Descripción:** Proporciona el listado detallado de todas las operaciones con información de embarcación, puerto, estado y fechas para su visualización en tablas.

**Tablas incluidas:** Operacion, EstadoOperacion, OperacionPortuaria, OperacionMaritima, Buque, Puerto, Muelle, TipoOperacionPortuaria, EstatusNavegacion

### Script de creación de vista

```sql
CREATE OR REPLACE VIEW gestion_maritima.v_listado_operaciones AS
SELECT 
    -- Información básica de la operación
    o.id_operacion,
    o.codigo AS codigo_operacion,
    o.fecha_inicio,
    o.fecha_fin,
    eo.nombre AS estado_operacion,
    
    -- Tipo de operación (Portuaria o Marítima)
    CASE 
        WHEN op.id_operacion_portuaria IS NOT NULL THEN 'Portuaria'
        WHEN om.id_operacion_maritima IS NOT NULL THEN 'Marítima'
        ELSE 'Sin clasificar'
    END AS tipo_operacion,
    
    -- Datos específicos de operación portuaria
    op.codigo AS codigo_operacion_portuaria,
    top.nombre AS tipo_operacion_portuaria,
    p.nombre AS puerto,
    p.codigo AS codigo_puerto,
    m.codigo AS codigo_muelle,
    m.ubicacion AS ubicacion_muelle,
    
    -- Datos específicos de operación marítima
    om.codigo AS codigo_operacion_maritima,
    om.cantidad_contenedores,
    om.porcentaje_trayecto,
    en.nombre AS estatus_navegacion,
    
    -- Información de la embarcación
    b.id_buque,
    b.matricula AS matricula_buque,
    b.nombre AS nombre_buque,
    b.capacidad AS capacidad_buque,
    b.peso AS peso_buque,
    b.ubicacion_actual AS ubicacion_actual_buque,
    eeb.nombre AS estado_embarcacion,
    
    -- Duración de la operación
    CASE 
        WHEN o.fecha_fin IS NOT NULL THEN 
            EXTRACT(EPOCH FROM (o.fecha_fin - o.fecha_inicio)) / 3600
        ELSE 
            EXTRACT(EPOCH FROM (CURRENT_TIMESTAMP - o.fecha_inicio)) / 3600
    END::DECIMAL(10,2) AS duracion_horas,
    
    -- Estado de finalización
    CASE 
        WHEN o.fecha_fin IS NOT NULL THEN TRUE
        ELSE FALSE
    END AS operacion_finalizada,
    
    -- Número de empleados asignados
    (SELECT COUNT(*) FROM gestion_maritima.OperacionEmpleado oe 
     WHERE oe.id_operacion = o.id_operacion 
     AND oe.fecha_desasignacion IS NULL) AS empleados_asignados,
    
    -- Número de contenedores asignados
    (SELECT COUNT(*) FROM gestion_maritima.OperacionContenedor oc 
     WHERE oc.id_operacion = o.id_operacion) AS contenedores_asignados,
    
    -- Número de incidencias
    (SELECT COUNT(*) FROM gestion_maritima.Incidencia i 
     WHERE i.id_operacion = o.id_operacion) AS total_incidencias,
    
    -- Número de inspecciones
    (SELECT COUNT(*) FROM gestion_maritima.Inspeccion ins 
     WHERE ins.id_operacion = o.id_operacion) AS total_inspecciones

FROM gestion_maritima.Operacion o
JOIN gestion_maritima.EstadoOperacion eo ON o.id_estado_operacion = eo.id_estado_operacion
LEFT JOIN gestion_maritima.OperacionPortuaria op ON o.id_operacion = op.id_operacion
LEFT JOIN gestion_maritima.TipoOperacionPortuaria top ON op.id_tipo_operacion_portuaria = top.id_tipo_operacion_portuaria
LEFT JOIN gestion_maritima.Puerto p ON op.id_puerto = p.id_puerto
LEFT JOIN gestion_maritima.Muelle m ON op.id_muelle = m.id_muelle
LEFT JOIN gestion_maritima.OperacionMaritima om ON o.id_operacion = om.id_operacion
LEFT JOIN gestion_maritima.EstatusNavegacion en ON om.id_estatus_navegacion = en.id_estatus_navegacion
LEFT JOIN gestion_maritima.Buque b ON COALESCE(op.id_buque, om.id_buque) = b.id_buque
LEFT JOIN gestion_maritima.EstadoEmbarcacion eeb ON b.id_estado_embarcacion = eeb.id_estado_embarcacion
ORDER BY o.fecha_inicio DESC;
```

## 3. Vista: v_registro_operacion_completo

**Pantalla:** P2 - Registro de Operación Portuaria y P6 - Registro de Operación Marítima

**Descripción:** Contiene toda la información necesaria para el formulario de registro y edición de operaciones, incluyendo catálogos de estados, tipos, puertos, muelles y resumen de contenedores y equipos.

**Tablas incluidas:** Operacion, EstadoOperacion, OperacionPortuaria, OperacionMaritima, Buque, Puerto, Muelle, TipoOperacionPortuaria, EstatusNavegacion, Contenedor, EquipoPortuario

### Script de creación de vista

```sql
CREATE OR REPLACE VIEW gestion_maritima.v_registro_operacion_completo AS
SELECT 
    -- Información de la operación
    o.id_operacion,
    o.codigo AS codigo_operacion,
    o.fecha_inicio,
    o.fecha_fin,
    o.id_estado_operacion,
    eo.nombre AS estado_operacion,
    
    -- Determinar tipo de operación
    CASE 
        WHEN op.id_operacion_portuaria IS NOT NULL THEN 'Portuaria'
        WHEN om.id_operacion_maritima IS NOT NULL THEN 'Marítima'
        ELSE NULL
    END AS tipo_operacion,
    
    -- Datos de operación portuaria
    op.id_operacion_portuaria,
    op.codigo AS codigo_operacion_portuaria,
    op.id_puerto,
    p.codigo AS codigo_puerto,
    p.nombre AS nombre_puerto,
    p.pais AS pais_puerto,
    op.id_muelle,
    m.codigo AS codigo_muelle,
    m.ubicacion AS ubicacion_muelle,
    m.capacidad AS capacidad_muelle,
    m.disponibilidad AS muelle_disponible,
    op.id_tipo_operacion_portuaria,
    top.nombre AS tipo_operacion_portuaria,
    
    -- Datos de operación marítima
    om.id_operacion_maritima,
    om.codigo AS codigo_operacion_maritima,
    om.cantidad_contenedores,
    om.porcentaje_trayecto,
    om.id_estatus_navegacion,
    en.nombre AS estatus_navegacion,
    
    -- Información del buque
    b.id_buque,
    b.matricula AS matricula_buque,
    b.nombre AS nombre_buque,
    b.capacidad AS capacidad_buque,
    b.peso AS peso_buque,
    b.ubicacion_actual,
    eeb.nombre AS estado_embarcacion,
    
    -- Resumen de contenedores
    (SELECT COUNT(*) FROM gestion_maritima.OperacionContenedor oc 
     WHERE oc.id_operacion = o.id_operacion) AS total_contenedores,
    (SELECT SUM(c.peso) FROM gestion_maritima.OperacionContenedor oc
     JOIN gestion_maritima.Contenedor c ON oc.id_contenedor = c.id_contenedor
     WHERE oc.id_operacion = o.id_operacion) AS peso_total_contenedores,
    
    -- Resumen de equipos portuarios (solo para operaciones portuarias)
    (SELECT COUNT(DISTINCT oep.id_equipo_portuario) 
     FROM gestion_maritima.OperacionEquipoPortuario oep
     WHERE oep.id_operacion_portuaria = op.id_operacion_portuaria
     AND oep.fecha_devolucion IS NULL) AS equipos_asignados,
    
    -- Resumen de empleados
    (SELECT COUNT(*) FROM gestion_maritima.OperacionEmpleado oe 
     WHERE oe.id_operacion = o.id_operacion 
     AND oe.fecha_desasignacion IS NULL) AS empleados_asignados,
    
    -- Información de ruta (para operaciones marítimas)
    orm.id_operacion_ruta_maritima,
    rm.id_ruta_maritima,
    rm.codigo AS codigo_ruta_maritima,
    r.codigo AS codigo_ruta,
    r.origen AS ruta_origen,
    r.destino AS ruta_destino,
    r.duracion AS duracion_ruta,
    r.tarifa AS tarifa_ruta,
    rm.distancia AS distancia_ruta,
    po.nombre AS puerto_origen,
    pd.nombre AS puerto_destino,
    mo.codigo AS codigo_muelle_origen,
    md.codigo AS codigo_muelle_destino,
    
    -- Certificaciones aduaneras (para operaciones marítimas)
    (SELECT COUNT(*) FROM gestion_maritima.OperacionCertificacionAduanera oca
     WHERE oca.id_operacion_maritima = om.id_operacion_maritima) AS total_certificaciones

FROM gestion_maritima.Operacion o
JOIN gestion_maritima.EstadoOperacion eo ON o.id_estado_operacion = eo.id_estado_operacion
LEFT JOIN gestion_maritima.OperacionPortuaria op ON o.id_operacion = op.id_operacion
LEFT JOIN gestion_maritima.Puerto p ON op.id_puerto = p.id_puerto
LEFT JOIN gestion_maritima.Muelle m ON op.id_muelle = m.id_muelle
LEFT JOIN gestion_maritima.TipoOperacionPortuaria top ON op.id_tipo_operacion_portuaria = top.id_tipo_operacion_portuaria
LEFT JOIN gestion_maritima.OperacionMaritima om ON o.id_operacion = om.id_operacion
LEFT JOIN gestion_maritima.EstatusNavegacion en ON om.id_estatus_navegacion = en.id_estatus_navegacion
LEFT JOIN gestion_maritima.Buque b ON COALESCE(op.id_buque, om.id_buque) = b.id_buque
LEFT JOIN gestion_maritima.EstadoEmbarcacion eeb ON b.id_estado_embarcacion = eeb.id_estado_embarcacion
LEFT JOIN gestion_maritima.OperacionRutaMaritima orm ON om.id_operacion_maritima = orm.id_operacion_maritima
LEFT JOIN gestion_maritima.RutaMaritima rm ON orm.id_ruta_maritima = rm.id_ruta_maritima
LEFT JOIN gestion_maritima.Ruta r ON rm.id_ruta = r.id_ruta
LEFT JOIN gestion_maritima.Puerto po ON rm.id_puerto_origen = po.id_puerto
LEFT JOIN gestion_maritima.Puerto pd ON rm.id_puerto_destino = pd.id_puerto
LEFT JOIN gestion_maritima.Muelle mo ON orm.id_muelle_origen = mo.id_muelle
LEFT JOIN gestion_maritima.Muelle md ON orm.id_muelle_destino = md.id_muelle;
```

## 4. Vista: v_selector_embarcacion

**Pantalla:** P3 - Seleccionar Embarcación

**Descripción:** Muestra el listado completo de embarcaciones con sus datos principales, estado operativo y disponibilidad para ser asignadas a operaciones.

**Tablas incluidas:** Buque, EstadoEmbarcacion, Operacion, OperacionPortuaria, OperacionMaritima

### Script de creación de vista

```sql
CREATE OR REPLACE VIEW gestion_maritima.v_selector_embarcacion AS
SELECT 
    -- Información básica del buque
    b.id_buque,
    b.matricula,
    b.nombre,
    b.capacidad,
    b.peso,
    b.ubicacion_actual,
    b.id_estado_embarcacion,
    ee.nombre AS estado_embarcacion,
    
    -- Disponibilidad operativa
    CASE 
        WHEN ee.nombre = 'Disponible' THEN TRUE  -- Cambiado de 'Operativo' a 'Disponible'
        ELSE FALSE
    END AS disponible,
    
    -- Operación actual (si está en uso)
    (SELECT o.id_operacion 
     FROM gestion_maritima.Operacion o
     LEFT JOIN gestion_maritima.OperacionPortuaria op ON o.id_operacion = op.id_operacion
     LEFT JOIN gestion_maritima.OperacionMaritima om ON o.id_operacion = om.id_operacion
     JOIN gestion_maritima.EstadoOperacion eo ON o.id_estado_operacion = eo.id_estado_operacion
     WHERE (op.id_buque = b.id_buque OR om.id_buque = b.id_buque)
     AND eo.nombre = 'En Progreso'  -- Cambiado de 'En curso' a 'En Progreso'
     ORDER BY o.fecha_inicio DESC
     LIMIT 1) AS id_operacion_actual,
    
    (SELECT o.codigo 
     FROM gestion_maritima.Operacion o
     LEFT JOIN gestion_maritima.OperacionPortuaria op ON o.id_operacion = op.id_operacion
     LEFT JOIN gestion_maritima.OperacionMaritima om ON o.id_operacion = om.id_operacion
     JOIN gestion_maritima.EstadoOperacion eo ON o.id_estado_operacion = eo.id_estado_operacion
     WHERE (op.id_buque = b.id_buque OR om.id_buque = b.id_buque)
     AND eo.nombre = 'En Progreso'  -- Cambiado de 'En curso' a 'En Progreso'
     ORDER BY o.fecha_inicio DESC
     LIMIT 1) AS codigo_operacion_actual,
    
    -- Certificaciones vigentes
    (SELECT COUNT(*) FROM gestion_maritima.CertificacionBuque cb
     WHERE cb.id_buque = b.id_buque
     AND cb.fecha_vencimiento > CURRENT_DATE) AS certificaciones_vigentes,
    
    -- Certificaciones vencidas o por vencer
    (SELECT COUNT(*) FROM gestion_maritima.CertificacionBuque cb
     WHERE cb.id_buque = b.id_buque
     AND cb.fecha_vencimiento <= CURRENT_DATE + INTERVAL '30 days') AS certificaciones_por_vencer,
    
    -- Historial de operaciones
    (SELECT COUNT(*) FROM gestion_maritima.Operacion o
     LEFT JOIN gestion_maritima.OperacionPortuaria op ON o.id_operacion = op.id_operacion
     LEFT JOIN gestion_maritima.OperacionMaritima om ON o.id_operacion = om.id_operacion
     WHERE (op.id_buque = b.id_buque OR om.id_buque = b.id_buque)) AS total_operaciones,
    
    (SELECT COUNT(*) FROM gestion_maritima.Operacion o
     LEFT JOIN gestion_maritima.OperacionPortuaria op ON o.id_operacion = op.id_operacion
     LEFT JOIN gestion_maritima.OperacionMaritima om ON o.id_operacion = om.id_operacion
     JOIN gestion_maritima.EstadoOperacion eo ON o.id_estado_operacion = eo.id_estado_operacion
     WHERE (op.id_buque = b.id_buque OR om.id_buque = b.id_buque)
     AND eo.nombre = 'Completada') AS operaciones_completadas,
    
    -- Última operación
    (SELECT MAX(o.fecha_fin) FROM gestion_maritima.Operacion o
     LEFT JOIN gestion_maritima.OperacionPortuaria op ON o.id_operacion = op.id_operacion
     LEFT JOIN gestion_maritima.OperacionMaritima om ON o.id_operacion = om.id_operacion
     WHERE (op.id_buque = b.id_buque OR om.id_buque = b.id_buque)
     AND o.fecha_fin IS NOT NULL) AS fecha_ultima_operacion,
    
    -- Capacidad utilizada (estimada)
    ROUND((b.peso / NULLIF(b.capacidad::DECIMAL, 0) * 100)::NUMERIC, 2) AS porcentaje_capacidad_base

FROM gestion_maritima.Buque b
JOIN gestion_maritima.EstadoEmbarcacion ee ON b.id_estado_embarcacion = ee.id_estado_embarcacion
ORDER BY b.nombre;
```

## 5. Vista: v_gestor_contenedores_operacion

**Pantalla:** P9 - Registro de Contenedores, y secciones de información de contenedores en P2 y P6

**Descripción:** Proporciona el listado completo de contenedores con sus características, estado, tipo y disponibilidad para ser asignados a operaciones.

**Tablas incluidas:** Contenedor, TipoContenedor, EstadoContenedor, OperacionContenedor, ContenedorMercancia

### Script de creación de vista

```sql
CREATE OR REPLACE VIEW gestion_maritima.v_gestor_contenedores_operacion AS
SELECT 
    -- Información básica del contenedor
    c.id_contenedor,
    c.codigo,
    c.peso,
    c.capacidad,
    c.dimensiones,
    
    -- Tipo de contenedor
    tc.id_tipo_contenedor,
    tc.codigo AS codigo_tipo,
    tc.nombre AS tipo_contenedor,
    tc.costo AS costo_tipo,
    
    -- Estado del contenedor
    ec.id_estado_contenedor,
    ec.nombre AS estado_contenedor,
    
    -- Porcentaje de carga
    ROUND((c.peso / NULLIF(c.capacidad, 0) * 100)::NUMERIC, 2) AS porcentaje_carga,
    
    -- Disponibilidad
    CASE 
        WHEN ec.nombre = 'Disponible' THEN TRUE
        ELSE FALSE
    END AS disponible,
    
    -- Mercancías contenidas
    (SELECT COUNT(*) FROM gestion_maritima.ContenedorMercancia cm
     WHERE cm.id_contenedor = c.id_contenedor) AS total_tipos_mercancia,
    
    (SELECT STRING_AGG(cm.tipo_mercancia, ', ')
     FROM gestion_maritima.ContenedorMercancia cm
     WHERE cm.id_contenedor = c.id_contenedor) AS tipos_mercancia,
    
    -- Operación actual (si está asignado)
    (SELECT o.id_operacion 
     FROM gestion_maritima.OperacionContenedor oc
     JOIN gestion_maritima.Operacion o ON oc.id_operacion = o.id_operacion
     JOIN gestion_maritima.EstadoOperacion eo ON o.id_estado_operacion = eo.id_estado_operacion
     WHERE oc.id_contenedor = c.id_contenedor
     AND eo.nombre IN ('En Progreso', 'En Planificación')  
     ORDER BY oc.fecha_asignacion DESC
     LIMIT 1) AS id_operacion_actual,
    
    (SELECT o.codigo 
     FROM gestion_maritima.OperacionContenedor oc
     JOIN gestion_maritima.Operacion o ON oc.id_operacion = o.id_operacion
     JOIN gestion_maritima.EstadoOperacion eo ON o.id_estado_operacion = eo.id_estado_operacion
     WHERE oc.id_contenedor = c.id_contenedor
     AND eo.nombre IN ('En Progreso', 'En Planificación')  
     ORDER BY oc.fecha_asignacion DESC
     LIMIT 1) AS codigo_operacion_actual,
    
    (SELECT oc.fecha_asignacion 
     FROM gestion_maritima.OperacionContenedor oc
     JOIN gestion_maritima.Operacion o ON oc.id_operacion = o.id_operacion
     JOIN gestion_maritima.EstadoOperacion eo ON o.id_estado_operacion = eo.id_estado_operacion
     WHERE oc.id_contenedor = c.id_contenedor
     AND eo.nombre IN ('En Progreso', 'En Planificación')  
     ORDER BY oc.fecha_asignacion DESC
     LIMIT 1) AS fecha_asignacion_actual,
    
    -- Historial de operaciones
    (SELECT COUNT(DISTINCT oc.id_operacion) 
     FROM gestion_maritima.OperacionContenedor oc
     WHERE oc.id_contenedor = c.id_contenedor) AS total_operaciones,
    
    -- Estiba actual (si aplica)
    (SELECT e.ubicacion 
     FROM gestion_maritima.Estiba e
```

```sql
CREATE OR REPLACE VIEW gestion_maritima.v_selector_rutas_maritimas AS
SELECT 
    -- Información de la ruta marítima
    rm.id_ruta_maritima,
    rm.codigo AS codigo_ruta_maritima,

    pd.id_puerto AS id_puerto_destino,
    pd.codigo AS codigo_puerto_destino,

    pd.pais AS pais_puerto_destino,

     WHERE rpi.id_ruta_maritima = rm.id_ruta_maritima) AS total_puertos_intermedios,
    
    (SELECT STRING_AGG(p.nombre, ' → ' ORDER BY p.nombre)

    -- Estimación de tiempo total
    CASE

INNER JOIN gestion_maritima.Puerto po ON rm.id_puerto_origen = po.id_puerto
INNER JOIN gestion_maritima.Puerto pd ON rm.id_puerto_destino = pd.id_puerto
```

```sql
     WHERE e2.id_operacion_portuaria = op.id_operacion_portuaria) AS peso_total_cargado,

            JOIN gestion_maritima.Contenedor c2 ON e2.id_contenedor = c2.id_contenedor
            WHERE e2.id_operacion_portuaria = op.id_operacion_portuaria)

FROM gestion_maritima.Estiba e

JOIN gestion_maritima.OperacionPortuaria op ON e.id_operacion_portuaria = op.id_operacion_portuaria
JOIN gestion_maritima.Operacion o ON op.id_operacion = o.id_operacion

JOIN gestion_maritima.Buque b ON op.id_buque = b.id_buque
```

## 9. Vista: v_contenedores_pendientes_estiba

### Script de creación de vista

```sql

    -- Información del contenedor
    c.id_contenedor,
    c.codigo AS codigo_contenedor,
    c.peso,
    c.capacidad,
    c.dimensiones,
    
    -- Tipo de contenedor
    tc.codigo AS codigo_tipo,
    tc.nombre AS tipo_contenedor,
    tc.costo,
    
    -- Estado
    ec.nombre AS estado_contenedor,
    
    -- Porcentaje de carga
    ROUND((c.peso / NULLIF(c.capacidad, 0) * 100)::NUMERIC, 2) AS porcentaje_carga,
    
    -- Mercancías
    (SELECT STRING_AGG(cm.tipo_mercancia, ', ')
     FROM gestion_maritima.ContenedorMercancia cm
     WHERE cm.id_contenedor = c.id_contenedor) AS tipos_mercancia,

    o.id_operacion,

    oc.fecha_asignacion,
    
    -- Información del buque
    b.id_buque,
    b.matricula AS matricula_buque,
    b.nombre AS nombre_buque,
    b.capacidad AS capacidad_buque,
    
    -- Espacio disponible en el buque
    (b.capacidad - COALESCE((SELECT SUM(c2.peso) 
                             FROM gestion_maritima.Estiba e
                             JOIN gestion_maritima.Contenedor c2 ON e.id_contenedor = c2.id_contenedor
                             WHERE e.id_operacion_portuaria = op.id_operacion_portuaria), 0)) AS espacio_disponible_buque,
    
    -- Contenedores ya estibados en esta operación
    (SELECT COUNT(*) 
     FROM gestion_maritima.Estiba e
     WHERE e.id_operacion_portuaria = op.id_operacion_portuaria) AS contenedores_ya_estibados

FROM gestion_maritima.Contenedor c
JOIN gestion_maritima.TipoContenedor tc 
    ON c.id_tipo_contenedor = tc.id_tipo_contenedor
JOIN gestion_maritima.EstadoContenedor ec

    ON op.id_buque = b.id_buque
```

## 14. Vista: v_gestor_incidencias_inspecciones

**Pantalla:** P15 - Operaciones con Incidencias y Registro de Inspección

**Descripción:** Muestra las operaciones que tienen incidencias registradas y proporciona información para el registro de nuevas inspecciones.

**Tablas incluidas:** Incidencia, TipoIncidencia, Operacion, EstadoOperacion, Usuario, Inspeccion, EstadoInspeccion, TipoInspeccion, Prioridad

### Script de creación de vista

```sql
CREATE OR REPLACE VIEW gestion_maritima.v_gestor_incidencias_inspecciones AS
SELECT 
    -- Información de la incidencia
    inc.id_incidencia,
    inc.codigo AS codigo_incidencia,
    inc.descripcion AS descripcion_incidencia,
    inc.grado_severidad,
    inc.fecha_hora AS fecha_hora_incidencia,
    
    -- Tipo de incidencia
    ti.id_tipo_incidencia,
    ti.nombre AS tipo_incidencia,
    
    -- Clasificación de severidad
    CASE 
        WHEN inc.grado_severidad = 5 THEN 'Crítica'
        WHEN inc.grado_severidad = 4 THEN 'Alta'
        WHEN inc.grado_severidad = 3 THEN 'Media'
        WHEN inc.grado_severidad = 2 THEN 'Baja'
        ELSE 'Mínima'
    END AS clasificacion_severidad,
    
    -- Información de la operación
    o.id_operacion,
    o.codigo AS codigo_operacion,
    o.fecha_inicio AS fecha_inicio_operacion,
    o.fecha_fin AS fecha_fin_operacion,
    eo.nombre AS estado_operacion,
    
    -- Tipo de operación
    CASE 
        WHEN op.id_operacion_portuaria IS NOT NULL THEN 'Portuaria'
        WHEN om.id_operacion_maritima IS NOT NULL THEN 'Marítima'
        ELSE 'Sin clasificar'
    END AS tipo_operacion,
    
    -- Usuario que registró la incidencia
    u.id_usuario,
    u.correo_electronico AS email_usuario,
    CONCAT(e.nombre, ' ', e.apellido) AS nombre_usuario,
    
    -- Inspecciones realizadas para esta incidencia
    (SELECT COUNT(*) 
     FROM gestion_maritima.Inspeccion ins
     WHERE ins.id_operacion = o.id_operacion
       AND ins.fecha >= inc.fecha_hora) AS inspecciones_posteriores,
    
    (SELECT COUNT(*) 
     FROM gestion_maritima.Inspeccion ins
     JOIN gestion_maritima.EstadoInspeccion ei 
       ON ins.id_estado_inspeccion = ei.id_estado_inspeccion
     WHERE ins.id_operacion = o.id_operacion
       AND ins.fecha >= inc.fecha_hora
       AND LOWER(ei.nombre) = 'completada') AS inspecciones_completadas,
    
    (SELECT COUNT(*) 
     FROM gestion_maritima.Inspeccion ins
     JOIN gestion_maritima.EstadoInspeccion ei 
       ON ins.id_estado_inspeccion = ei.id_estado_inspeccion
     WHERE ins.id_operacion = o.id_operacion
       AND ins.fecha >= inc.fecha_hora
       AND LOWER(ei.nombre) = 'pendiente') AS inspecciones_pendientes,
    
    -- Última inspección
    (SELECT MAX(ins.fecha) 
     FROM gestion_maritima.Inspeccion ins
     WHERE ins.id_operacion = o.id_operacion) AS fecha_ultima_inspeccion,
    
    -- Hallazgos relacionados
    (SELECT COUNT(*) 
     FROM gestion_maritima.Inspeccion ins
     JOIN gestion_maritima.Hallazgo h 
       ON ins.id_inspeccion = h.id_inspeccion
     WHERE ins.id_operacion = o.id_operacion
       AND ins.fecha >= inc.fecha_hora) AS total_hallazgos,
    
    -- Requiere inspección
    CASE 
        WHEN (
            SELECT COUNT(*) 
            FROM gestion_maritima.Inspeccion ins
            WHERE ins.id_operacion = o.id_operacion
              AND ins.fecha >= inc.fecha_hora
        ) = 0 
        AND inc.grado_severidad >= 3 THEN TRUE
        ELSE FALSE
    END AS requiere_inspeccion,
    
    -- Tiempo transcurrido desde la incidencia
    EXTRACT(EPOCH FROM (CURRENT_TIMESTAMP - inc.fecha_hora)) / 3600::DECIMAL(10,2) AS horas_desde_incidencia,
    
    -- Información del buque
    b.matricula AS matricula_buque,
    b.nombre AS nombre_buque,
    
    -- Total de incidencias en la operación
    (SELECT COUNT(*) 
     FROM gestion_maritima.Incidencia i2
     WHERE i2.id_operacion = o.id_operacion) AS total_incidencias_operacion,
    
    -- Promedio de severidad en la operación
    (SELECT AVG(i2.grado_severidad)::DECIMAL(3,2)
     FROM gestion_maritima.Incidencia i2
     WHERE i2.id_operacion = o.id_operacion) AS severidad_promedio_operacion

FROM gestion_maritima.Incidencia inc
JOIN gestion_maritima.TipoIncidencia ti 
  ON inc.id_tipo_incidencia = ti.id_tipo_incidencia
JOIN gestion_maritima.Operacion o 
  ON inc.id_operacion = o.id_operacion
JOIN gestion_maritima.EstadoOperacion eo 
  ON o.id_estado_operacion = eo.id_estado_operacion
JOIN gestion_maritima.Usuario u 
  ON inc.id_usuario = u.id_usuario
JOIN gestion_maritima.Empleado e 
  ON u.id_empleado = e.id_empleado
LEFT JOIN gestion_maritima.OperacionPortuaria op 
  ON o.id_operacion = op.id_operacion
LEFT JOIN gestion_maritima.OperacionMaritima om 
  ON o.id_operacion = om.id_operacion
LEFT JOIN gestion_maritima.Buque b 
  ON COALESCE(op.id_buque, om.id_buque) = b.id_buque
ORDER BY inc.grado_severidad DESC, inc.fecha_hora DESC;
```

```sql
    CASE 
        WHEN op.id_operacion_portuaria IS NOT NULL THEN 'Portuaria'
        WHEN om.id_operacion_maritima IS NOT NULL THEN 'Marítima'
        ELSE 'Sin clasificar'
    END AS tipo_operacion,
    
    -- Inspector (usuario)
    u.id_usuario AS id_inspector,
    CONCAT(e.nombre, ' ', e.apellido) AS nombre_inspector,
    u.correo_electronico AS email_inspector,
    e.codigo AS codigo_empleado_inspector,
    esp.nombre AS especialidad_inspector,
    e.anios_experiencia AS experiencia_inspector,
    
    -- Hallazgos
    (SELECT COUNT(*) FROM gestion_maritima.Hallazgo h
     WHERE h.id_inspeccion = ins.id_inspeccion) AS total_hallazgos,
    
    (SELECT COUNT(*) FROM gestion_maritima.Hallazgo h
     WHERE h.id_inspeccion = ins.id_inspeccion
       AND h.nivel_gravedad >= 4) AS hallazgos_criticos,
    
    (SELECT COUNT(*) FROM gestion_maritima.Hallazgo h
     WHERE h.id_inspeccion = ins.id_inspeccion
       AND h.nivel_gravedad = 3) AS hallazgos_moderados,
    
    (SELECT COUNT(*) FROM gestion_maritima.Hallazgo h
     WHERE h.id_inspeccion = ins.id_inspeccion
       AND h.nivel_gravedad <= 2) AS hallazgos_leves,
    
    -- Gravedad máxima encontrada
    (SELECT MAX(h.nivel_gravedad) 
     FROM gestion_maritima.Hallazgo h
     WHERE h.id_inspeccion = ins.id_inspeccion) AS gravedad_maxima,
    
    -- Promedio de gravedad
    (SELECT AVG(h.nivel_gravedad)::DECIMAL(3,2) 
     FROM gestion_maritima.Hallazgo h
     WHERE h.id_inspeccion = ins.id_inspeccion) AS gravedad_promedio,
    
    -- Tipos de hallazgos encontrados
    (SELECT STRING_AGG(DISTINCT th.nombre, ', ')
     FROM gestion_maritima.Hallazgo h
     JOIN gestion_maritima.TipoHallazgo th 
       ON h.id_tipo_hallazgo = th.id_tipo_hallazgo
     WHERE h.id_inspeccion = ins.id_inspeccion) AS tipos_hallazgos,
    
    -- Hallazgos con acciones pendientes
    (SELECT COUNT(*) FROM gestion_maritima.Hallazgo h
     WHERE h.id_inspeccion = ins.id_inspeccion
       AND h.accion_sugerida IS NOT NULL
       AND LENGTH(TRIM(h.accion_sugerida)) > 0) AS hallazgos_con_acciones,
    
    -- Tiempo transcurrido desde la inspección
    EXTRACT(EPOCH FROM (CURRENT_TIMESTAMP - ins.fecha)) / 3600::DECIMAL(10,2) AS horas_desde_inspeccion,
    
    -- Días desde la inspección
    EXTRACT(DAY FROM (CURRENT_TIMESTAMP - ins.fecha))::INT AS dias_desde_inspeccion,
    
    -- Incidencias relacionadas
    (SELECT COUNT(*) FROM gestion_maritima.Incidencia inc
     WHERE inc.id_operacion = o.id_operacion
       AND inc.fecha_hora <= ins.fecha) AS incidencias_previas,
    
    (SELECT COUNT(*) FROM gestion_maritima.Incidencia inc
     WHERE inc.id_operacion = o.id_operacion
       AND inc.fecha_hora > ins.fecha) AS incidencias_posteriores,
    
    -- Información del buque (si aplica)
    b.id_buque,
    b.matricula AS matricula_buque,
    b.nombre AS nombre_buque,
    b.capacidad AS capacidad_buque,
    eeb.nombre AS estado_buque,
    
    -- Información del puerto (si es operación portuaria)
    p.id_puerto,
    p.codigo AS codigo_puerto,
    p.nombre AS nombre_puerto,
    p.pais AS pais_puerto,
    
    -- Información del muelle (si es operación portuaria)
    m.id_muelle,
    m.codigo AS codigo_muelle,
    m.ubicacion AS ubicacion_muelle,
    
    -- Estado de completitud (lookup ajustado con LOWER)
    CASE 
        WHEN LOWER(ei.nombre) = 'completada' THEN TRUE
        ELSE FALSE
    END AS inspeccion_completada,
    
    -- Requiere acción urgente
    CASE 
        WHEN (SELECT MAX(h.nivel_gravedad) 
              FROM gestion_maritima.Hallazgo h
              WHERE h.id_inspeccion = ins.id_inspeccion) >= 4 
             AND LOWER(ei.nombre) != 'completada' THEN TRUE
        ELSE FALSE
    END AS requiere_accion_urgente,
    
    -- Indicadores adicionales
    CASE 
        WHEN (SELECT COUNT(*) 
              FROM gestion_maritima.Hallazgo h
              WHERE h.id_inspeccion = ins.id_inspeccion) = 0 THEN 'Sin hallazgos'
        WHEN (SELECT MAX(h.nivel_gravedad) 
              FROM gestion_maritima.Hallazgo h
              WHERE h.id_inspeccion = ins.id_inspeccion) >= 4 THEN 'Crítico'
        WHEN (SELECT MAX(h.nivel_gravedad) 
              FROM gestion_maritima.Hallazgo h
              WHERE h.id_inspeccion = ins.id_inspeccion) = 3 THEN 'Moderado'
        ELSE 'Leve'
    END AS clasificacion_inspeccion,
    
    -- Información concatenada para búsqueda
    CONCAT(ins.codigo, ' - ', ti.nombre, ' - ', ei.nombre) AS inspeccion_completa

FROM gestion_maritima.Inspeccion ins
JOIN gestion_maritima.TipoInspeccion ti 
  ON ins.id_tipo_inspeccion = ti.id_tipo_inspeccion
JOIN gestion_maritima.EstadoInspeccion ei 
  ON ins.id_estado_inspeccion = ei.id_estado_inspeccion
JOIN gestion_maritima.Prioridad pr 
  ON ins.id_prioridad = pr.id_prioridad
JOIN gestion_maritima.Operacion o 
  ON ins.id_operacion = o.id_operacion
JOIN gestion_maritima.EstadoOperacion eo 
  ON o.id_estado_operacion = eo.id_estado_operacion
JOIN gestion_maritima.Usuario u 
  ON ins.id_usuario = u.id_usuario
JOIN gestion_maritima.Empleado e 
  ON u.id_empleado = e.id_empleado
JOIN gestion_maritima.EspecialidadEmpleado esp 
  ON e.id_especialidad_empleado = esp.id_especialidad_empleado
LEFT JOIN gestion_maritima.OperacionPortuaria op 
  ON o.id_operacion = op.id_operacion
LEFT JOIN gestion_maritima.OperacionMaritima om 
  ON o.id_operacion = om.id_operacion
LEFT JOIN gestion_maritima.Buque b 
  ON COALESCE(op.id_buque, om.id_buque) = b.id_buque
LEFT JOIN gestion_maritima.EstadoEmbarcacion eeb 
  ON b.id_estado_embarcacion = eeb.id_estado_embarcacion
LEFT JOIN gestion_maritima.Puerto p 
  ON op.id_puerto = p.id_puerto
LEFT JOIN gestion_maritima.Muelle m 
  ON op.id_muelle = m.id_muelle;

```
