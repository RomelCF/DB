# 9.2.3. Módulo 3 / Integrante 3


## Módulo de Gestión de Operaciones Marítimas 

### Funcionalidad primaria del Módulo de Gestión de Operaciones Marítimas

#### Flujo elegido: Registro y Planificación de una Operación Marítima de Carga


- Descripción del flujo: El flujo inicia en el *Dashboard de Operaciones Marítimas (P05)*, donde el usuario crea una nueva operación marítima y selecciona una embarcación (P03). Luego se verifica si la capacidad o peso de la embarcación es suficiente; si no lo es, se debe modificar y volver a seleccionar. Si cumple, se procede al registro de contenedores (P09), donde se seleccionan y confirman los contenedores a transportar. Si la operación no tiene una ruta preasignada, se definen las rutas marítimas (P07 y P08) hasta confirmar una, generando una vista preliminar de la operación (P06). Posteriormente, se revisan y planifican los datos; si es necesario, pueden modificarse la embarcación, contenedores o ruta. Luego se realiza la asignación de tripulación y se valida su disponibilidad. Una vez que la tripulación está asignada, se registra la operación marítima como carga planificada (P02) y se evalúa si la planificación está lista o aprobada. Si aún requiere ajustes, se modifican horarios o muelle; de lo contrario, se hace clic en “Iniciar Operación”, marcando el inicio de la operación marítima en curso.

-----

#### Justificación de la Elección

La elección del flujo de Registro y Planificación de una Operación Marítima de Carga  como funcionalidad primaria se basa en las siguientes razones:

1.  **Integración de Datos Clave:** Este flujo obliga a la integración y uso de información proveniente de varias "tablas" o módulos clave del negocio marítimo: Embarcaciones (P03), Contenedores (P09), Rutas (P07, P08), Equipos Portuarios (P04), y Detalles de la Operación (P02, P06).
2.  **Proceso Central del Negocio:** La planificación de una operación de carga (o descarga/estiba) es una actividad de alto valor y el corazón de la logística de un carrier como Hapag-Lloyd, conectando la estrategia comercial (rutas) con la ejecución operativa (terminal/buque).
3.  **No CRUD:** Cumple con la restricción de no enfocarse en operaciones CRUD simples (crear/modificar/eliminar un registro único), sino en un proceso completo de planificación que culmina con una acción transaccional importante ("Iniciar Operación").
4.  **Consulta/Reporte Implícito:** La interfaz final (P02 y P06) actúa como una ficha de reporte y consulta consolidada de la operación planificada, mostrando todos los elementos seleccionados (embarcación, contenedores, equipos, personal, ruta, tiempos).

---

#### Mapeo de Requisitos Funcionales

| **Requerimiento** | **Implementado en el Flujo** |
| -- | -- |
| RF03 | Se selecciona y valida la embarcación (capacidad/peso suficiente) durante la configuración inicial.|
| RF05 | Al finalizar el flujo, la operación pasa al estado “Operación Marítima En Curso” |
| RF06 | Es el núcleo del flujo: se registran datos de embarcación, ruta marítima, tripulación y contenedores asociados. |
| RF07 | En la etapa de definición de ruta (P07 y P08) se buscan y seleccionan rutas marítimas.|
| RF08 | Se registran y confirman contenedores asociados a la operación (P09), con posibilidad de modificarlos. |


#### Interfaces de Usuario Incluidas

Pantallas Principales Implementadas:


- **P03 - Listado de embarcaciones disponibles con filtros y búsqueda. (CU03)**
    - Buscar embarcaciones por nombre o matrícula.

    - Filtrar embarcaciones por rango de peso o capacidad.

    - Seleccionar una embarcación disponible para la operación.

    - Confirmar la selección de la embarcación.

- **P05 - Dashboard de operaciones marítimas activas con información de progreso. (CU05)**
    - Buscar operaciones por código o buque.

    - Filtrar operaciones por Estatus Navegación (Navegando, En puerto, En tránsito).

    - Ver el porcentaje de Trayecto Completado.

    - Crear una Nueva Operación Marítima o Actualizar el Dashboard.

- **P06 - Menú de registro de operación marítima con embarcación, ruta, tripulación y contenedores. (CU06)**
    - Verificar o cambiar la embarcación y la ruta asignada.

    - Asignar tripulación (Capitán, Jefe de Máquinas, Primer Oficial).

    - Establecer los detalles de la operación (Estado, Hora Estimada, Muelle de origen/destino).

    - Iniciar la Operación (o Cancelarla). 

- **P07 - Mapa interactivo para selección de rutas marítimas con filtros. (CU07)**
    - Buscar rutas por Puerto de Origen, Puerto de Destino y Rango de fechas.

    - Visualizar la ruta activa en el mapa y su leyenda.

    - Ver información de la ruta (Origen, Destino, Distancia Total, Tiempo Estimado).

    - Confirmar la ruta seleccionada o Limpiar mapa. 

- **P08 - Listado de rutas disponibles según criterios de búsqueda. (CU07)**
    - Visualizar rutas entre el puerto de origen y destin.
    - Comparar detalles de la ruta: distancia, duración, puertos intermedios y tarifa.
    - Seleccionar una ruta.
    - Confirmar la ruta seleccionada.

- **P09 - Listado de contenedores reservados pendientes para una ruta marítima dada. (CU08)**
    - Filtrar contenedores por ruta.

    - Visualizar el estado actual de cada contenedor (En Tránsito, En Puerto, En Aduana, Retenido).

    - Seleccionar contenedores para la operación (ej: CON-930217).

    - Ver detalles como Peso, Capacidad, Dimensiones y Tipo de mercancía.


##### **Consultas Principales:**

## 1. **Panel de Operaciones en Curso**
```sql
SELECT 
    op.codigo AS codigo_operacion,
    opm.codigo AS codigo_operacion_maritima,
    eo.nombre AS estado_operacion,
    en.nombre AS estatus_navegacion,
    opm.cantidad_contenedores,
    opm.porcentaje_trayecto,
    b.matricula,
    b.nombre AS nombre_buque,
    op.fecha_inicio,
    op.fecha_fin
FROM gestion_maritima.Operacion op
INNER JOIN gestion_maritima.OperacionMaritima opm ON op.id_operacion = opm.id_operacion
INNER JOIN gestion_maritima.EstadoOperacion eo ON op.id_estado_operacion = eo.id_estado_operacion
INNER JOIN gestion_maritima.EstatusNavegacion en ON opm.id_estatus_navegacion = en.id_estatus_navegacion
INNER JOIN gestion_maritima.Buque b ON opm.id_buque = b.id_buque
WHERE op.fecha_fin IS NULL OR op.fecha_fin >= CURRENT_DATE
ORDER BY op.fecha_inicio DESC;
```

## 2. **Búsqueda de Rutas por Criterios**
```sql
SELECT 
    rm.codigo AS codigo_ruta,
    r.codigo AS codigo_ruta_base,
    po.nombre AS puerto_origen,
    pd.nombre AS puerto_destino,
    rm.distancia,
    r.duracion,
    r.tarifa,
    r.origen,
    r.destino,
    (
        SELECT COUNT(*)
        FROM gestion_maritima.RutaPuertoIntermedio rpi
        WHERE rpi.id_ruta_maritima = rm.id_ruta_maritima
    ) AS cantidad_puertos_intermedios
FROM gestion_maritima.RutaMaritima rm
INNER JOIN gestion_maritima.Ruta r ON rm.id_ruta = r.id_ruta
INNER JOIN gestion_maritima.Puerto po ON rm.id_puerto_origen = po.id_puerto
INNER JOIN gestion_maritima.Puerto pd ON rm.id_puerto_destino = pd.id_puerto
WHERE po.nombre = 'Nombre Puerto Origen'
AND pd.nombre = 'Nombre Puerto Destino'
AND rm.id_ruta_maritima NOT IN (
    SELECT orm.id_ruta_maritima 
    FROM gestion_maritima.OperacionRutaMaritima orm
    INNER JOIN gestion_maritima.OperacionMaritima om ON orm.id_operacion_maritima = om.id_operacion_maritima
    INNER JOIN gestion_maritima.Operacion o ON om.id_operacion = o.id_operacion
    WHERE o.fecha_inicio BETWEEN 'fecha_inicio' AND 'fecha_fin'
);
```

## 3. **Detalles Completos de Ruta**
```sql
SELECT 
    rm.codigo AS codigo_ruta_maritima,
    r.codigo AS codigo_ruta,
    po.nombre AS puerto_origen,
    pd.nombre AS puerto_destino,
    rm.distancia,
    r.duracion,
    r.tarifa,
    mo.codigo AS muelle_origen,
    md.codigo AS muelle_destino,
    ARRAY_AGG(pi.nombre) AS puertos_intermedios
FROM gestion_maritima.RutaMaritima rm
INNER JOIN gestion_maritima.Ruta r ON rm.id_ruta = r.id_ruta
INNER JOIN gestion_maritima.Puerto po ON rm.id_puerto_origen = po.id_puerto
INNER JOIN gestion_maritima.Puerto pd ON rm.id_puerto_destino = pd.id_puerto
LEFT JOIN gestion_maritima.RutaPuertoIntermedio rpi ON rm.id_ruta_maritima = rpi.id_ruta_maritima
LEFT JOIN gestion_maritima.Puerto pi ON rpi.id_puerto = pi.id_puerto
LEFT JOIN gestion_maritima.Muelle mo ON po.id_puerto = mo.id_puerto AND mo.disponibilidad = true
LEFT JOIN gestion_maritima.Muelle md ON pd.id_puerto = md.id_puerto AND md.disponibilidad = true
WHERE rm.codigo = 'codigo_ruta_seleccionada'
GROUP BY rm.codigo, r.codigo, po.nombre, pd.nombre, rm.distancia, r.duracion, r.tarifa, mo.codigo, md.codigo;
```

## 4. **Buques Disponibles por Capacidad**
```sql
SELECT 
    b.matricula,
    b.nombre,
    b.capacidad,
    b.peso,
    b.ubicacion_actual,
    ee.nombre AS estado_embarcacion,
    COUNT(cb.id_certificacion) AS certificaciones_activas
FROM gestion_maritima.Buque b
INNER JOIN gestion_maritima.EstadoEmbarcacion ee ON b.id_estado_embarcacion = ee.id_estado_embarcacion
LEFT JOIN gestion_maritima.CertificacionBuque cb ON b.id_buque = cb.id_buque 
    AND cb.fecha_vencimiento >= CURRENT_DATE
WHERE ee.nombre = 'Disponible'
AND b.capacidad >= capacidad_requerida
AND b.peso >= peso_requerido
AND b.id_buque NOT IN (
    SELECT opm.id_buque 
    FROM gestion_maritima.OperacionMaritima opm
    INNER JOIN gestion_maritima.Operacion o ON opm.id_operacion = o.id_operacion
    WHERE o.id_estado_operacion IN (
        SELECT id_estado_operacion 
        FROM gestion_maritima.EstadoOperacion 
        WHERE nombre IN ('En Progreso', 'En Planificación')
    )
)
GROUP BY b.id_buque, b.matricula, b.nombre, b.capacidad, b.peso, b.ubicacion_actual, ee.nombre
ORDER BY b.capacidad DESC;
```

## 5. **Contenedores Disponibles por Tipo**
```sql
SELECT 
    c.codigo,
    tc.nombre AS tipo_contenedor,
    c.peso,
    c.capacidad,
    c.dimensiones,
    ec.nombre AS estado_contenedor,
    tc.costo,
    cm.tipo_mercancia
FROM gestion_maritima.Contenedor c
INNER JOIN gestion_maritima.TipoContenedor tc ON c.id_tipo_contenedor = tc.id_tipo_contenedor
INNER JOIN gestion_maritima.EstadoContenedor ec ON c.id_estado_contenedor = ec.id_estado_contenedor
LEFT JOIN gestion_maritima.ContenedorMercancia cm ON c.id_contenedor = cm.id_contenedor
WHERE ec.nombre = 'Disponible'
AND c.id_contenedor NOT IN (
    SELECT oc.id_contenedor 
    FROM gestion_maritima.OperacionContenedor oc
    INNER JOIN gestion_maritima.Operacion o ON oc.id_operacion = o.id_operacion
    WHERE o.id_estado_operacion IN (
        SELECT id_estado_operacion 
        FROM gestion_maritima.EstadoOperacion 
        WHERE nombre IN ('En Progreso', 'En Planificación')
    )
)
AND tc.codigo IN ('T-001', 'T-002', 'T-003')
ORDER BY tc.costo DESC, c.capacidad DESC;
```

## 6. **Asignación de Contenedores a Operación**
```sql
-- Para cada contenedor seleccionado
UPDATE gestion_maritima.Contenedor 
SET id_estado_contenedor = (
    SELECT id_estado_contenedor 
    FROM gestion_maritima.EstadoContenedor 
    WHERE nombre = 'En Tránsito'
)
WHERE codigo = 'codigo_contenedor';

-- Insertar en OperacionContenedor
INSERT INTO gestion_maritima.OperacionContenedor (
    id_operacion_contenedor, 
    id_operacion, 
    id_contenedor, 
    fecha_asignacion
)
SELECT 
    gen_random_uuid(),
    (SELECT id_operacion FROM gestion_maritima.Operacion WHERE codigo = 'codigo_operacion'),
    id_contenedor,
    CURRENT_DATE
FROM gestion_maritima.Contenedor 
WHERE codigo = 'codigo_contenedor';
```

## 7. **Configuración Básica de Operación**
```sql
UPDATE gestion_maritima.OperacionMaritima 
SET id_buque = (
    SELECT id_buque 
    FROM gestion_maritima.Buque 
    WHERE matricula = 'matricula_buque'
)
WHERE id_operacion = (
    SELECT id_operacion 
    FROM gestion_maritima.Operacion 
    WHERE codigo = 'codigo_operacion'
);

INSERT INTO gestion_maritima.OperacionRutaMaritima (
    id_operacion_ruta_maritima,
    id_operacion_maritima,
    id_ruta_maritima,
    id_muelle_origen,
    id_muelle_destino
)
SELECT 
    gen_random_uuid(),
    om.id_operacion_maritima,
    rm.id_ruta_maritima,
    (SELECT id_muelle FROM gestion_maritima.Muelle WHERE codigo = 'muelle_origen'),
    (SELECT id_muelle FROM gestion_maritima.Muelle WHERE codigo = 'muelle_destino')
FROM gestion_maritima.OperacionMaritima om
INNER JOIN gestion_maritima.RutaMaritima rm ON rm.codigo = 'codigo_ruta'
WHERE om.id_operacion = (
    SELECT id_operacion 
    FROM gestion_maritima.Operacion 
    WHERE codigo = 'codigo_operacion'
);
```

## 8. **Completar Detalles de Operación**
```sql
-- Actualizar operación con fechas
UPDATE gestion_maritima.Operacion 
SET 
    fecha_inicio = 'fecha_inicio_estimada',
    fecha_fin = 'fecha_fin_estimada'
WHERE codigo = 'codigo_operacion';

-- Asignar tripulación
INSERT INTO gestion_maritima.OperacionEmpleado (
    id_operacion_empleado,
    id_operacion,
    id_empleado,
    fecha_asignacion
)
SELECT 
    gen_random_uuid(),
    (SELECT id_operacion FROM gestion_maritima.Operacion WHERE codigo = 'codigo_operacion'),
    id_empleado,
    CURRENT_DATE
FROM gestion_maritima.Empleado 
WHERE codigo IN ('codigo_empleado1', 'codigo_empleado2');
```

## 9. **Creación Completa de Operación**
```sql
-- Primero crear la operación base
INSERT INTO gestion_maritima.Operacion (
    id_operacion,
    codigo,
    fecha_inicio,
    id_estado_operacion
)
VALUES (
    gen_random_uuid(),
    'codigo_operacion_nuevo',
    CURRENT_TIMESTAMP,
    (SELECT id_estado_operacion FROM gestion_maritima.EstadoOperacion WHERE nombre = 'En Planificación')
)
RETURNING id_operacion;

-- Luego crear la operación marítima
INSERT INTO gestion_maritima.OperacionMaritima (
    id_operacion_maritima,
    id_operacion,
    codigo,
    cantidad_contenedores,
    id_estatus_navegacion,
    porcentaje_trayecto,
    id_buque
)
SELECT 
    gen_random_uuid(),
    id_operacion,
    'OM-' || codigo,
    cantidad_contenedores,
    (SELECT id_estatus_navegacion FROM gestion_maritima.EstatusNavegacion WHERE nombre = 'En Puerto'),
    0.00,
    (SELECT id_buque FROM gestion_maritima.Buque WHERE matricula = 'matricula_buque')
FROM gestion_maritima.Operacion 
WHERE codigo = 'codigo_operacion_nuevo';
```

## 10. **Actualización de Estado de Buque**
```sql
UPDATE gestion_maritima.Buque 
SET id_estado_embarcacion = (
    SELECT id_estado_embarcacion 
    FROM gestion_maritima.EstadoEmbarcacion 
    WHERE nombre = 'En operación'
)
WHERE matricula = 'matricula_buque';
```
