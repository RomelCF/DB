**Eventos:**

- **Cargar Lista de Buques:**

   Al cargar la pantalla, se muestran todos los buques y la cantidad de resutados.
   
```sql
SELECT 
    b.id_buque,
    b.nombre,
    b.matricula,
    b.capacidad,
    e.nombre AS estado
FROM Buque b
INNER JOIN EstadoEmbarcacion e
    ON b.id_estado_embarcacion = e.id_estado_embarcacion;
```

- **Cargar Lista de Buques Disponibles:**

   Al hacer click en el boton "Disponible" este filtrará la tabla de buques y mostrará la cantidad de resultados.
   
```sql
SELECT 
    b.id_buque,
    b.nombre,
    b.matricula,
    b.capacidad
FROM Buque b
INNER JOIN EstadoEmbarcacion e
    ON b.id_estado_embarcacion = e.id_estado_embarcacion
WHERE e.nombre = 'Disponible';
```

```sql
SELECT 
    COUNT(*) AS disponibles
FROM Buque b
INNER JOIN EstadoEmbarcacion e
    ON b.id_estado_embarcacion = e.id_estado_embarcacion
WHERE e.nombre = 'Disponible';

```
 - **Cargar Lista de Buques no disponibles:**

    Al hacer click en el boton "No disponible" este filtrará la tabla de buques y mostrará la cantidad de resultados.

```sql
SELECT 
    b.id_buque,
    b.nombre,
    b.matricula,
    e.nombre AS estado
FROM Buque b
INNER JOIN EstadoEmbarcacion e
    ON b.id_estado_embarcacion = e.id_estado_embarcacion
WHERE e.nombre <> 'Disponible';
```

```sql
SELECT 
    COUNT(*) AS no_disponibles
FROM Buque b
INNER JOIN EstadoEmbarcacion e
    ON b.id_estado_embarcacion = e.id_estado_embarcacion
WHERE e.nombre <> 'Disponible';
```

**Eventos**

- **Cargar Lista de Tripulantes**

    Al cargar la pantalla, se muestran a los tripulantes y la cantidad de resutados.

```sql
SELECT 
    t.id_tripulante,
    e.nombre,
    e.apellido,
    e.dni,
    t.nacionalidad,
    t.disponibilidad
FROM Tripulante t
INNER JOIN Empleado e 
    ON t.id_empleado = e.id_empleado
WHERE t.disponibilidad = TRUE;
```
```sql
SELECT COUNT(*) AS disponibles
FROM Tripulante
WHERE disponibilidad = TRUE
```

- **Detalle de cada Tripulante**

    Al hacer click en los usuario debe mostrar un panel donde esta los certificados del tripulante

```sql
SELECT 
    c.nombre AS certificado,
    ce.fecha_obtencion,
    ce.fecha_vencimiento
FROM CertificacionEmpleado ce
INNER JOIN Certificacion c 
    ON ce.id_certificacion = c.id_certificacion
WHERE ce.id_empleado = (
    SELECT id_empleado
    FROM Tripulante
    WHERE id_tripulante = ?   -- aquí el tripulante clickeado
);
```
```sql
SELECT 
    COUNT(*) AS cantidad_certificados
FROM CertificacionEmpleado ce
WHERE ce.id_empleado = (
    SELECT id_empleado
    FROM Tripulante
    WHERE id_tripulante = ?   -- aquí el mismo tripulante clickeado
);
```

- **Boton de Asignación**

    Al hacer click en el boton "Asignar Tripulantes" agrega los tripulantes al buque y cambia su estado

```sql
INSERT INTO BuqueTripulante (
    id_buque,
    id_tripulante,
    fecha_asignacion
) VALUES (
    ?,           -- id del buque seleccionado
    ?,           -- id del tripulante marcado
    CURDATE()    -- fecha actual
);
```

```sql
UPDATE Tripulante
SET disponibilidad = FALSE
WHERE id_tripulante = ?;
```

**Evento**

- **Vizualizar historial de asignacion**

    - Se mostrara el historial de las asignaciones de cierto buque previamente elegido

```sql
SELECT 
    t.id_tripulante,
    e.nombre AS nombre_tripulante,
    e.apellido,
    e.dni,
    t.nacionalidad,
    ce.nombre AS rol,
    bt.fecha_asignacion
FROM BuqueTripulante bt
INNER JOIN Tripulante t ON bt.id_tripulante = t.id_tripulante
INNER JOIN Empleado e ON t.id_empleado = e.id_empleado
LEFT JOIN CertificacionEmpleado ce ON ce.id_empleado = e.id_empleado
WHERE bt.id_buque = ?;
```
